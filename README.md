## ✨ Why This Portfolio Stands Out

This isn't just a standard resume website. It is designed to be a **premium digital experience** that reflects a deep passion for Artificial Intelligence, Agentic Systems, and top-tier Software Engineering. 

- 🎨 **Luxury UI/UX:** A stunning dark mode and light mode toggle with carefully selected typography and color palettes.
- 🪄 **Micro-Interactions:** Custom cursors, magnetic buttons, and scroll-triggered animations powered by Framer Motion.
- ⚡ **Blazing Fast:** Server-side rendered (SSR) with Next.js 14 App Router for instant load times and perfect SEO.
- 🧠 **AI-Focused:** Sections tailored specifically to highlight machine learning, LLMs, and prompt engineering skills.

## 🚀 Getting Started (Local Development)

Want to see how it works under the hood? It's incredibly easy to run locally.

```bash
# 1. Clone the repository
git clone https://github.com/priyachaudhary4/priya-portfolio.git

# 2. Navigate into the project
cd priya-portfolio

# 3. Install the dependencies
npm install

# 4. Start the development server
npm run dev
```

Open **[http://localhost:3000](http://localhost:3000)** to view it in the browser!
