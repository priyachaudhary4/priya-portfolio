## ✨ Features

- **Modern & Premium Design:** A sleek, luxury aesthetic with dark/light mode support.
- **Dynamic Animations:** Smooth transitions and micro-interactions powered by Framer Motion.
- **Custom Cursor:** An interactive custom cursor that adapts to hover states.
- **Fully Responsive:** Optimized for all screen sizes, from mobile to desktop.
- **Performance Focused:** Built with Next.js App Router for optimal speed and SEO.

## 🛠️ Tech Stack

This project is built using modern web technologies to ensure high performance and a great developer experience:

- **Framework:** [Next.js 14](https://nextjs.org/) (React)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Deployment:** [Vercel](https://vercel.com/)

## 🏃‍♀️ Running Locally

To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/priyachaudhary4/priya-portfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd priya-portfolio
   ```
3. Install dependencies:
   ```bash
   npm install
   # or
   pnpm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   # or
   pnpm dev
   ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser.
